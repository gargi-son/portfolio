EasySplit About

  The main focus of this project is to analyze human behavior and further Human-Computer Interaction research by exposing users to new application prototypes, logging their reactions, and improving the design prototype. Easy-split is one such finance management application. It is developed with the intention to help users resolve the problem of equitable expense divisions by managing and tracking their expenses better. This web application is UI-focused, with the front-end framework designed using the ReactJS library with embedded logic to split the expenses, and can be extended by integrating Firebase to support the applicationâ€™s database needs.
